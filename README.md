### Features

- [x] Clickable buttons!
- [x] Clean, lightweight, and themable UI
- [x] Clear instructions
- [x] Optional timer
- [x] Make and manage multiple presences
- [x] Custom, simplified developer portal
- [x] Easy installers for Windows, macOS, and Linux
